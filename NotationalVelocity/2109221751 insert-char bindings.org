Some of the chars that are useful have direct bindings all starting with *C-x 8* so it isn't neccessary to type in the name. *which-key* doesn't seem to know about these although they do (mostly) work anyway. There is a list at: http://ergoemacs.org/emacs/gnu_emacs_keybinding_C-x_8.txt (Gone, but [[https://web.archive.org/web/20191223021553/http://ergoemacs.org:80/emacs/gnu_emacs_keybinding_C-x_8.txt][archive]] has it. Looks ok in 2019 capture)

Some highlights:
°   *C-x 8 * o*
©   *C-x 8 C*

½   *C-x 8 1 / 2*
¼   *C-x 8 1 / 4*
¾   *C-x 8 3 / 4*

²   *C-x 8 ^ 2*
³   *C-x 8 ^ 3*
