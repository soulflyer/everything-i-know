#+TITLE: use-package
# Tags #emacs

Each package has an *ensure* statement 
#+begin_src emacs-lisp
  (use-package blah
    :ensure t)
#+end_src
This could be avoided by setting a variable:
#+begin_src emacs-lisp
  (setq use-package-always-ensure t)
#+end_src
