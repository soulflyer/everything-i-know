Passing -c/--create-frame to emacsclient should do the trick.

If I run 
#+begin_src shell
emacsclient -c --eval "(magit-status \"/path/to/project\")" 
#+end_src
I end up with a new emacs frame in a magit-status buffer for the expected project.
