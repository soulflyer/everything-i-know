M-x describe-char
 will give all the info you might want about a char at point.

C-u C-x = 
Calls *what-cursor-position* with a parameter which does something similar

Use M-x insert-char followed by name or hex unicode value. (C-x 8 RET)

more info here: https://emacs.stackexchange.com/a/61388/2666

Use of unicode/utf-8 on POSIX systems. Lots of useful info: https://www.cl.cam.ac.uk/~mgk25/unicode.html
Current Unicode standard (2021): https://www.unicode.org/versions/Unicode14.0.0/
